# Script for chads

### Onboarding
[[essential_dev_ubuntu.sh]]:
 - Install essential development tools on Ubuntu 
 - git, tmux, eza, unzip, build-essential, neovim, NvChad, OhMyPosh
 - Architecture sensitive.
 - Set alias for ls, ll, la to use eza
 - Usage: `curl -fsSL https://raw.githubusercontent.com/wongwaijeremy/scripts/master/essential_dev_ubuntu.sh | bash`
